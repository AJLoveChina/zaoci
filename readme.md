### 我爱造新词
我爱造新词 (微信小程序) 是年轻人的一种娱乐方式

代码完全开源, 可以随意使用, 不依赖任何服务端图片文件, 开箱即用.

### 为什么要有dev分支?
因为小程序的审核过于严格, 他们不允许生成的海报页面包含二维码. 那好, 我也有办法, 我通过一个服务端配置文件来控制是否显示二维码, 提交审核时我先关闭二维码, 审核通过后我再开启二维码. 机智~~
